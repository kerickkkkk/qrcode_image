# vue qrcode 中間有 logo 

用給圖片網址就把這個圖片擺在中間。
目前主要基於 [vue-qrcode](https://github.com/fengyuanchen/vue-qrcode) 實作，使用 canvas 將圖片直接畫在 canvas 上。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
